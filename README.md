## ✨ 核心特性
- 基于 [Hugo](https://gohugo.io/) 静态网站生成器
- 响应式设计，完美适配移动端
- GitHub Flavored Markdown 支持
- 自动生成文章目录
- Prism 代码高亮
- 多语言支持（中/英）
- GitHub Actions 自动部署
- RSS/Atom 订阅支持
